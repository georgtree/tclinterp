Tcl wrapper for C interpolation procedures (v0.1)

ContentTop, Main, Index

This package provides tcl wrapper for interpolation procedures. The source of procedures is Interpolation Routines.

Installation and dependenciesTop, Main, Index

To install this package just unzip code to folder and append auto_path variable with location.

For compile you need:

  • SWIG of version 4.3
  • gcc compiler
  • Tcl9 stubs object file libtclstub.a - usually located at /usr/local/lib/, but location could differ on your system

For run you need:

  • Tcl9
  • argparse
  • Tcllib

To compile, run swig_gen.sh to create SWIG wrapper file, and the run make. If you have stubs object file in non-standard location or with different name, add it to Makefile:

Then the shared file will be located at /build folder, the remaining part is to add tclinterp to auto_path.
