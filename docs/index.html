Tcl wrapper for C interpolation procedures (v0.1)

ContentTop, Main, Index

This package provides tcl wrapper for interpolation procedures. The source of procedures is Interpolation Routines.

Installation and dependenciesTop, Main, Index

For building you need:

  • SWIG of version 4.3
  • Tcl9
  • gcc compiler
  • make tool
  • autoconf tool

For run you also need:

  • argparse
  • Tcllib

To compile, run swig_gen.sh to create SWIG wrapper file, then run

For Windows build it is strongly recommended to use MSYS64 UCRT64 environment, the above steps are identical if you run it from UCRT64 shell. After installing the package, you can move tclinterp package folder (usually located in C:\msys64\ucrt64\lib\) to path listed in auto_path variable of your local Tcl installation.

Supported platformsTop, Main, Index

I've tested it on:

  • Kubuntu 24.04 with Tcl9
  • Windows 11 in MSYS64 UCRT64 environment with Tcl9
